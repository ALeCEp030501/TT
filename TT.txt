Общие сведения
	Мобильное приложение для предоставления информации о курсе различных валют - Licure
	Заказчик - 
	Начало работы 29.10.22
	Конец работы 30.11.22

Цели и назначение создания приложения
	Цель Licure - показать текущую информацию о валютах, выбранных пользователями.
	Назначение - просмотр информации

Характеристика объекта автоматизации
	Данные о курсах валют, приложение должно получать с сайтов https://myfin.by 
								   https://www.xe.com/currencyconverter/.
	
Требования к Licure
	Приложение должно быть написано на языке JAVA, при использовании IDE Android Studio.
	При написании кода, должна быть соответсвующая к нему документация, для изменений в случае необходимости.
	Приложение работает только при подключении к интернету, проверяет данные на сайте при каждом запуске.
	Данные, полученные с сайта должны сохраняться в локальную базу данных на смартфоне,
	и храниться там до следующего обновления данных на сайте.
	Обновление базы данных производиться по необходимости при каждом запуске, либо по желанию пользователя вручную, 
	либо при работающем приложении происходит проверка данных на сайте каждые 30 минут и, 
	при наличии обновлений, обновлять соответствующую информацию.
	Должна быть возможность выбора источников информации, а также предоставлена возможность сравнения 
	курсов валют из разных источников
	Приложение должно поддерживать языки всех стран СНГ.
	Licure должен поддерживаться устройствами Android версией 5.0 и выше.
	Интерфейс Licure должен иметь светлую и тёмную темы.
	Светлая тема должны быть в сочетании синих и белых цветов.
	Тёмная тема должна быть в сочетании синих и тёмных цветов.
	При проектировании интерфейса использовать готовые шаблоны IDE Android Studio.

Требования к документированию
	Исходный код должен быть полностью описан.
	Это подразумевает описание каждой функии, каждого класса, а также логика работы программы.
	Должны быть включены документы, названия книг и ссылки на скачивание либо на просмотр их, а также ссылки на использовавшиеся вами источники информации
